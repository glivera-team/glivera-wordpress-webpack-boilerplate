* ACF 5

== Installation ==

1. Copy the `acf-mtd-extends` folder into your `wp-content/plugins` folder
2. Activate the `ACF: MTD Extend Fields` plugin via the plugins admin page
3. Create a new field via ACF and select the `MTD Contact Form 7` type
4. Read the description above for usage instructions
